# Markdown + LaTeX 混合渲染修复 Demo

## 📖 项目说明

这是一个展示如何使用 Kiro AI Agent 解决 Markdown 加粗语法（`**`）与 LaTeX 数学公式冲突问题的在线演示页面。

## 🚀 快速开始

### 本地预览

1. 直接用浏览器打开 `index.html` 文件即可查看效果

### 部署到 GitHub Pages

1. 将整个项目推送到 GitHub 仓库

2. 进入仓库的 Settings → Pages

3. 在 "Source" 下选择：
   - Branch: `main` (或 `master`)
   - Folder: `/demo` 或 `/ (root)`

4. 点击 Save，等待几分钟后访问：
   ```
   https://你的用户名.github.io/仓库名/
   ```

### 部署到 Vercel（推荐）

1. 访问 [vercel.com](https://vercel.com)

2. 点击 "Import Project"

3. 选择你的 GitHub 仓库

4. 设置 Root Directory 为 `demo`

5. 点击 Deploy，几秒钟后即可获得在线链接
