✅ 修复后（正常渲染）

1. 时间膨胀公式

当一个物体以速度 \(v\) 相对于观察者运动时：

\[t = \frac{t_0}{\sqrt{1-\frac{v^{2}}{c^{2}}}}\]

其中 \(t_0\) 是固有时，\(v\) 是运动速度，\(c\) 是光速。

✅ 修复后

洛伦兹变换

- \(x\) 坐标变换：\(x'=\gamma(x - vt)\)

- 时间变换：\(t' = \gamma(t - \frac{v}{c^{2}}x)\)

其中 \(\gamma=\frac{1}{\sqrt{1 - \frac{v^{2}}{c^{2}}}}\)

💡 技术方案

核心思路：预处理 + 双重防护

1. 预处理阶段

// 在 Markdown 解析前清理公式内外的 ** 标记 private static _stripStarsAroundFormulas(text: string): string { // 剥掉包裹块级公式的 ** text = text.replace(/\*\*(\s*\\\[[\s\S]*?\\\]\s*)\*\*/g, '$1'); // 剥掉 $...$ 行内公式内部的 ** text = text.replace(/\$([^$]*?)\$/g, (match, inner) => { return '$' + inner.replace(/\*\*/g, '') + '$'; }); return text; }

2. 解析阶段防护

// 查找 ** 时跳过公式内部 private static findStarOutsideMath(text, startIndex): number { while (i < text.length) { // 跳过 $...$ 公式 if (text[i] === '$') { /* 跳过逻辑 */ } // 跳过 \(...\) 公式 if (text[i] === '\\' && text[i+1] === '(') { /* 跳过 */ } // 跳过 \[...\] 公式 if (text[i] === '\\' && text[i+1] === '[') { /* 跳过 */ } if (text[i] === '*') return i; } }
